# Как пользоваться HTML Presentation Editor

## Быстрый старт

1. Открой файл:

```
editor/presentation-editor-v12.html
```

в браузере (Chrome / Edge / Firefox)

2. Нажми кнопку **«Открыть HTML»**

3. Выбери один из вариантов:

* загрузить `.html` файл
* вставить HTML код вручную

После загрузки редактор автоматически:

* найдёт слайды
* построит список слева
* откроет первый слайд

---

## Работа со слайдами

### Переключение

* слева отображается список слайдов
* клик по слайду → открывается в центре

### Порядок

* стрелки ↑ ↓ → изменить порядок слайдов

---

## Редактирование контента

### Основной принцип

Редактирование происходит **прямо в слайде**

1. Кликни по тексту (заголовок, абзац, список)
2. Начни редактировать

**Важно:**

* изменения применяются сразу
* ничего сохранять не нужно

---

## Настройка текста

Выбери элемент → справа появится панель настроек

Доступно:

* Жирный / курсив / подчёркнутый
* Размер шрифта
* Выравнивание текста
* Цвет текста

Все изменения применяются мгновенно через inline-стили

---

## Работа с HTML

### Редактирование элемента

Кнопка:

```
Показать HTML элемента
```

* открывается окно с HTML
* можно отредактировать вручную
* после сохранения изменения применяются

---

### Редактирование слайда

Кнопка:

```
Показать HTML слайда
```

* открывается HTML всего слайда
* можно полностью заменить разметку
* после сохранения слайд обновляется

---

## Экспорт

Нажми:

```
Экспортировать HTML
```

Результат:

* скачивается файл:

```
presentation-edited.html
```

* в нём уже все изменения

---

## Требования к HTML

Слайды должны быть оформлены как:

```
<section class="slide">...</section>
```

или:

```
<div class="slide">...</div>
```

---

## Важные нюансы

### 1. Стили и ресурсы

Если используются:

* CSS
* изображения

лучше:

* использовать абсолютные ссылки
  или
* хранить рядом с HTML

---

### 2. Скрипты

JavaScript внутри презентации:

* не выполняется (это сделано для безопасности)

---

### 3. Сохранение

* редактор не хранит состояние
* всё сохраняется только через экспорт

---

## Типовой сценарий использования

1. Сгенерировать HTML-презентацию (например, через ИИ)
2. Открыть редактор
3. Загрузить HTML
4. Отредактировать:

   * тексты
   * заголовки
   * формулировки
5. При необходимости поправить HTML
6. Экспортировать финальный файл

---

## Итог

Редактор позволяет:

* быстро править HTML-презентации
* не работать с кодом напрямую
* получать готовый файл для использования
