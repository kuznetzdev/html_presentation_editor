# HTML Presentation Editor

Локальное одностраничное веб-приложение для визуального редактирования HTML-презентаций без необходимости работать с кодом.

Поддерживает загрузку готовых HTML-слайдов (например, как в ) и редактирование в реальном времени.

---

## 1. Назначение

Редактор предназначен для работы с HTML-презентациями, которые уже сгенерированы (например, через ИИ или вручную).

Основной сценарий:

* загрузить HTML-файл презентации
* отредактировать текст и стили визуально
* при необходимости — поправить HTML точечно
* экспортировать обновлённый файл

---

## 2. Ключевые возможности

### Работа с HTML

* загрузка `.html` файла с локального компьютера
* вставка HTML кода вручную
* сохранение полного HTML документа после редактирования

### Работа со слайдами

* автоматическое определение слайдов:

  ```js
  const SLIDE_SELECTOR = 'section.slide';
  ```
* fallback:

  ```js
  div.slide
  ```
* список слайдов с заголовками
* переключение между слайдами
* изменение порядка (вверх / вниз)

### Визуальное редактирование

* редактирование текста прямо в слайде (contenteditable)
* изменения применяются **в реальном времени**
* подсветка элементов при hover / focus

### Форматирование

* жирный / курсив / подчёркнутый
* размер шрифта
* выравнивание
* цвет текста

### Работа с HTML

* просмотр HTML элемента
* ручная правка HTML элемента
* просмотр HTML всего слайда
* замена DOM слайда через HTML

### Экспорт

* сборка полного HTML документа
* скачивание файла:

  ```
  presentation-edited.html
  ```

---

## 3. Архитектура

### Подход

* без backend
* чистый:

  * HTML
  * CSS
  * JavaScript

### Ключевая идея

Редактор работает через:

```
iframe (srcdoc) + DOM синхронизация
```

Это позволяет:

* сохранять оригинальные стили презентации
* редактировать «как в реальном браузере»
* не ломать верстку

### Безопасность

* iframe sandbox:

  ```
  sandbox="allow-same-origin"
  ```
* скрипты внутри загруженного HTML **не выполняются**

---

## 4. Структура интерфейса

### Левая панель

* список слайдов
* номер + заголовок
* активный слайд подсвечен

### Центральная область

* живое превью слайда
* редактирование inline

### Правая панель

* свойства элемента
* форматирование
* доступ к HTML

### Верхняя панель

* открыть HTML
* показать HTML слайда
* экспорт

---

## 5. Как использовать

### 1. Открытие

Открыть файл:

```
presentation-editor.html
```

в браузере (Chrome / Edge / Firefox)

---

### 2. Загрузка презентации

Варианты:

* загрузить файл `.html`
* вставить HTML код

После загрузки:

* строится список слайдов
* открывается первый слайд

---

### 3. Редактирование

1. Выбрать слайд слева
2. Кликнуть по тексту
3. Редактировать напрямую

Особенности:

* изменения применяются сразу
* ничего отдельно сохранять не нужно

---

### 4. Настройка стилей

Через правую панель:

* размер текста
* цвет
* выравнивание
* форматирование

Все изменения:

```
inline CSS (element.style)
```

---

### 5. Ручная правка HTML

Два уровня:

#### Элемент

* кнопка: «Показать HTML элемента»

#### Слайд

* кнопка: «Показать HTML слайда»

После сохранения:

* DOM обновляется

---

### 6. Экспорт

Нажать:

```
Экспортировать HTML
```

Результат:

* полный HTML документ
* файл скачивается

---

## 6. Ограничения

### Ресурсы (CSS / картинки)

Если используются относительные пути:

* должны лежать рядом с HTML
  или
* использовать абсолютные URL

---

### JavaScript внутри презентации

* не выполняется (sandbox)

---

### execCommand

Используется для форматирования:

* работает в большинстве браузеров
* fallback реализован через style

## 9. Пример входного HTML

Слайды должны быть в формате:

```html
<section class="slide">
  ...
</section>
```

или:

```html
<div class="slide">
  ...
</div>
```
