Открыть HTML

Для максимально точного превью презентация загружается как отдельный документ в iframe, а её скрипты не вырезаются. Используй этот режим только для доверенных HTML‑файлов.

Загрузка файла

Выбери локальный HTML‑файл презентации. Обычно это presentation.html.

Вставка HTML

Подходит для сценария, когда HTML презентации уже сгенерирован и лежит в буфере обмена.

Папка ресурсов

Если у презентации есть относительные css/js/img/video, выбери папку проекта или assets. Это улучшит честность превью.

Вставить видео

Локальный файл

Подходит для коротких роликов. Для переносимого HTML лучше держать файл умеренного размера.

Практический лимит для self-contained HTML: до 12 МБ.

Ссылка на видео

Поддерживаются YouTube, Vimeo и прямые ссылки на mp4/webm/ogg. Можно вставить и относительный путь из папки проекта.

GIF вставляй через «Картинка / GIF», чтобы он оставался обычным изображением.
Для больших видео лучше использовать прямую ссылку на mp4/webm или держать файл рядом с HTML и вставлять относительный путь.
