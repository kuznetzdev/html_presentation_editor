1-minute local start

  1. Make sure Node.js 18+ is available.
  2. Run npm start in the repo root.
  3. Open http://127.0.0.1:4173/.
  4. Click Open Editor.
  5. Use the sample gallery if you want a ready-made deck to upload first.
npm start
