Simple start

Open HTML Presentation Editor

This repository already contains the app. Start the local server, open the editor, load an HTML deck, and edit it visually. You do not need to inspect the repo first.

Open Editor Browse Example Decks

How to start

  1. Run npm start or double-click start-editor.cmd.
  2. Open http://127.0.0.1:4173/.
  3. Click Open Editor and load your HTML deck.

What to open first

  • Open Editor opens the active runtime.
  • Browse Example Decks gives you ready-made sample files.
  • The stable local URL is /.

Runtime path

editor/presentation-editor.html

Run npm start or double-click start-editor.cmd if you just want the app to open locally without thinking about repo internals.
