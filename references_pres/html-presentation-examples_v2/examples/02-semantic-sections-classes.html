Пример 02 • semantic sections + classes

Внутренний CSS и переиспользуемая структура

Этот вариант уже ближе к реальному проекту: семантическая разметка, понятные классы и повторяющиеся UI-паттерны. Подходит для редактора как более реалистичный тест структуры контейнеров.

Идея

Слайды остаются обычными section.slide, но оформление переносится в stylesheet, а HTML становится заметно чище.

Блоки и карточки

Читаемость

Код проще сопровождать, потому что повторяемость уехала в CSS-классы.

Системность

Легче тестировать тему, размеры, отступы и типографику.

Редактируемость

Редактор может менять текст, карточки, порядок и стили без inline-шума.
