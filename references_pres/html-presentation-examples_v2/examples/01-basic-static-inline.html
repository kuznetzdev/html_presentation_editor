Пример 01 • inline styles

HTML-презентация без CSS-файлов и библиотек

Этот пример показывает самый прямой способ написания HTML deck: обычные <section>, вся стилизация живёт прямо в тегах, без JS и без внешних зависимостей.

3 слайда
0 внешних файлов
1 HTML-документ

Когда такой способ полезен

  • Быстрый прототип без структуры проекта.
  • Тесты редактора на простом single-file HTML.
  • Проверка базового текста, блоков и цветов.
Плюсы Максимально прозрачно, легко читать, удобно для ручной правки.
Минусы Нет переиспользуемых классов, быстро разрастается и сложнее сопровождать.

Что проверить в редакторе

  1. Выбор текста и контейнеров.
  2. Редактирование inline-стилей.
  3. Дублирование, удаление и экспорт single-file deck.
