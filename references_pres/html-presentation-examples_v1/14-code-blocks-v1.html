Code block contract

Whitespace-safe editing for imported snippets

Блок кода должен редактироваться как plain-text, не теряя отступы, переводы строк и author classes. 

const reviewer = "ops";
function openDeck() {
  return reviewer;
}
Что важно сохранить
1. Корневой <pre> и вложенный <code>.
2. Отступы второй строки.
3. CSS-классы для syntax/highlight пайплайна.
