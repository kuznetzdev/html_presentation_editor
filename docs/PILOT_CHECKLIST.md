# Pilot Checklist

## Before inviting users

- `npm install` completed on the pilot machine
- `npm run test:install-browsers` completed once
- `npm run test:pilot` passes
- Root launchpad opens from `/`
- `Open Starter Example` loads successfully
- A text edit can be made in edit mode
- Exported HTML opens outside the editor
- The operator has read `docs/KNOWN_LIMITATIONS.md`

## User A

- Opens starter example
- Understands the preview-first flow without explanation
- Makes one text edit
- Exports successfully

## User B

- Opens a manual deck file
- Can duplicate a slide
- Can undo and redo
- Exports successfully

## Pilot log

- Record browser and OS
- Record whether asset-folder import was used
- Record whether any step required operator intervention
- Record whether the user understood the save/export distinction
