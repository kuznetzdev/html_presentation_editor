# Структура репозитория

## editor/
Актуальный файл редактора.

- `presentation-editor-v0.18.1.html` — текущая рабочая сборка (v0.18.1)

## docs/
Основная проектная документация.

- `SOURCE_OF_TRUTH.md` — главный документ проекта
- `PROJECT_SUMMARY.md` — сжатое описание продукта, стадии и стратегии
- `ROADMAP_NEXT.md` — последовательность следующих инженерных этапов
- `CODEX_HANDOFF_PROMPTS.md` — готовые промпты для Codex
- `REPORT_V12.md` — отчёт по v12
- `VALIDATION_NOTES_V12.md` — заметки по проверке v12
- `REMAINING_ISSUES.md` — backlog после v12

## docs/history/
История инженерных проходов.

Содержит:
- предыдущие HTML-версии
- diff между версиями
- отчёты
- validation notes
- previous remaining issues

## docs/source-materials/
Исходные ТЗ, handoff и сопутствующие материалы, от которых велась разработка.
